# PW-Generator
 A simple GUI based Password-Generator

DE:

PW-Generator ist ein GUI basierter Passwort-Generator der verschiedenste Funktionen bietet, um ein Passwort zu generieren. Zurzeit sind 2 Versionen verfuegbar. Das ist zum einen die "Customtkinter Version", die mit
einem moderneren Design daherkommt. Die zweite Version ist die "only tkinter version". Sie hat die normalen Tkinter Sachen (also Buttons etc.). Du kannst alle älteren Versionen auch aus dem Ordner "old" Downloaden!


ENG:

PW-Generator is a gui based password generator. The generator have many functions to create a password. For now are 2 versions available, the "customtkinter version" and the "only tkinter version". The custom
tkinter version has a newer GUI. The only tkinter version have the standard things (like Buttons etc.). The older Versionen are in the "old" folder, you can Download all Files!
